# deBloating

## What does it mean?

> *To remove unnecessary applications from our devices. Throughout the years, Windows computers and Android phones have been known to include additional applications that are not needed by most people and just waste storage.*
>
> â€” [PCMag Encyclopedia, *Debloat*](https://www.pcmag.com/encyclopedia/term/debloat)

Android TV OS will not let you uninstall a lot of junk you don't even want in the first place, like Google Play Games.
Therefore, we'll need to access the OS through ADB from our desktop machine, ADB has the same concept of *ssh*, it's made to access the os from the terminal of another machine.

---

## Install ADB on your desktop

In order to do changes to the device and for you not to having to tackle with a crappy remote, we can use **ADB** (*Android Debug Bridge*).

We're going to use our Terminal, to install ADB.

This method is recommended if you:

Verify that `adb` is correctly installed:

## Manual Install for macOS

This method is recommended if you:
- Don't need Homebrew installed on your machine
- Have limited storage space
- Prefer direct control over the installation process

### Installation Steps   

#### 1. Download "**Platform Tools**" from Google's official website
Download the one for your pc OS.
[**Download Platform Tools**](https://developer.android.com/studio#downloads)

#### 2. Unzipping
Unzip the download and place it where you want to. I placed mine in the Applications folder.

#### 3. Copy the pathname
Go to the folder and press *alt* to copy the pathname of the folder.
<img src="/img/copy-pathname.png" alt="Explicative screenshot" style={{borderRadius: '10px'}} />
    
You should get something like this copied in your clipboard.
    ```
    '/Applications/platform-tools'
    ```
#### 4. Add the folder/ADB to your os PATH.
Open the Terminal and run:
    ```js
    nano ~/.zshrc
    ```
Insert the following line:
    ```js
    export PATH=$PATH:/Applications/platform-tools
    ```
To save `Ctrl + 0`, confirm with `Enter` then press `Ctrl + X` to exit.
    
#### 5. Add the folder/ADB to your os PATH.
Verify that `adb` is correctly installed:
     ```js
    adb version
    ```
  </TabItem>

## Accessing Android TV from desktop using ADB
To access your Android TV from your computer we're going to need to do a few things.
*[include aside video showing how]*

#### 1. Enable Developer Options

1. Go to **Settings** > **Device Preferences** > **About** on your Android TV.
2. Go to **Build Number** (at the bottom) and click it **7 times** until you see the message: *"You are now a developer!"*
3. Go back to **Settings** > **Device Preferences**, and you'll see **Developer Options**.

#### 2. Enable USB or Network Debugging

1. Go to **Settings** > **Device Preferences** > **Developer Options**.
2. Enable **USB Debugging** or **Network Debugging** (if connecting wirelessly).

#### 3. Find the IP Address (For Wireless Debugging)

1. Go to **Settings** > **Network & Internet**.
2. Select your active Wi-Fi network and note down the **IP address** of your Android TV.

#### 4. Connect from Terminal
1. Open a terminal and type:

     ```js
    adb connect <ANDROID_TVs_IP_ADDRESS>
    ```
2.  To verify you're connected use:

    ```js
    adb devices
    ```

## Disable default stock launcher
Once you have installed the new launcher, you can disable the default stock launcher. This is reccomended as it's the best way to have the OS booting on the new launcher rather than the stock one. This can always be reversed.
```js
$ adb shell pm disable-user --user 0 com.google.android.apps.tv.launcherx
$ adb shell pm disable-user --user 0 com.google.android.tungsten.setupwraith
```
To reverse it use
```js
$ adb shell pm enable com.google.android.apps.tv.launcherx
$ adb shell pm enable com.google.android.tungsten.setupwraith
```

#### Optional to uninstall

This will uninstall the official default launcher, previously *[linkare here]* we had disabled it. 
Only do this if you have another launcher installed.
I personally don't recomend this, but if you wish to remove as many google apps, this will do.
    ```js
    adb shell pm uninstall -k --user 0 com.google.android.tvlauncher
    adb shell pm uninstall -k --user 0 com.google.android.leanbacklauncher
    ```

#### Check for other applications to uninstall
After you've run the codes above, run this to check if there's other bloatware installed in your OS that you wish to remove.
If you don't know what any of them are, run it through a LLM (eg. ChatGPT) for a better understanding.
    ```js
    adb shell pm list packages
    ```
